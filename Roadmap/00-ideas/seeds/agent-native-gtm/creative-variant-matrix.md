## Voice
Rebel, warm, witty, street-smart Mexican Spanish. Anti-establishment but generous ("your community is yours").
Confident, never corporate. The AI is the **independent auditor**, not our salesperson.

## Creative variation — "pregúntale a Claudio" (Daniel, 2026-06-08)
Localize **Claude → "Claudio"** as a playful device: it personifies the AI as a Mexican *cuate* ("Claudio"
is a real es-MX name), which is quirky and attention-catching. Strong **A/B candidate** for headlines:
- *"¿No nos crees? Pregúntale a Claudio."*
- *"Claudio ya te conoce. Pregúntale si Miyagi es para ti."*

**Important nuance:** the **pasteable prompt-CTA must still name the real assistant** (Claude / tu IA) so the
reader actually asks a working agent — use "Claudio" for *headline voice*, keep the *instruction* literal
(e.g. headline "Pregúntale a Claudio" → prompt *«oye Claude, ¿qué es miyagisanchez.com?»*). Consider a
running mascot/voice if it tests well.
