# Own-shop premium presentation — "stepping into StickerJunkie"

*Seeded 2026-07-03 while grooming `custom-print-products` (Daniel chose to split this thread out).*

**Raw ask:** walking into `miyagiprints.miyagisanchez.com` (or any tenant's own domain/subdomain)
should feel like walking into stickerjunkie.com — a real brand's own store, not a marketplace
profile page. Build the premium feeling of owning your shop on your own domain, reusable by every
tenant.

**Audit findings to start from (2026-07-03):** the *plumbing* is done — Diseño y marca (accent,
socials, tagline), Canal propio (custom domain / subdomain / embed / support widget), white-label
`ChannelLayout`, slug + short links, trust parity strip. The gap is **storefront presentation**:

- Shop **hero/banner** (image + value line, like SJ's "FREE SHIPPING | RUSH AVAILABLE" bar)
- **In-shop navigation** — categories/collections within one shop (SJ's product-type menu), not one
  flat listing grid
- **Product-type landing pages** per shop (all die-cut, all zines…)
- Richer **theming** — typography/background beyond one accent color; per-shop OG cards
- Shop-level **reviews/social proof** surface
- Content blocks: FAQ / About / policies rendered on the storefront (some data already exists in
  settings — Devoluciones — but isn't merchandised)

**Reference end-states (inspiration, never signed-off scope):** stickerjunkie.com, stickermule.com.
