# 06 · Print Edition — *"Sal en la edición impresa"*

**The marketplace's first and only revenue channel.** A free, ad-funded local print magazine with a México-86 retro aesthetic. Local businesses pay to be featured; QR codes and WhatsApp bridge the printed page back to the marketplace. Miyagi designs every ad — that's the value-add; advertisers just bring the ingredients.

Think of it as three connected experiences:

1. **Buy a placement** *(advertiser-facing)* — a business picks an ad size, builds their ad, and pays.
2. **Community / social section** *(reader-facing)* — anyone can submit recommendations, events, and shout-outs.
3. **Make the magazine** *(editor-facing)* — Miyagi reviews submissions, lays out the issue, and sends a print-ready file to the press.

## Current features
- ✅ **Ad placements** in tiers (full / half / quarter / card), sold like any product
- ✅ **Self-serve ad builder** for advertisers (copy, photos, contact, QR target) with draft + pay
- ✅ **Community / social section** — recommendations, recognitions, events, greetings
- ✅ **Editorial queue** — review, approve/reject, request changes
- ✅ **Advertiser self-service** — manage ads, see status, report manual payments, request changes
- ✅ **Production export pack** — a ZIP for the designer
- ✅ **Layout builder ("Maqueta")** + **one-click print-ready PDF** → see the epic below
